## ğŸš€ Quick Start

### 1. Installation
1. **Download** from [Modrinth](https://modrinth.com/plugin/spectator-plus-plus)
2. **Place** the `.jar` file in your server's `plugins/` folder
3. **Restart** your server

### 2. Basic Setup
1. **Set spectator lobby** (essential):/spectator lobby set
2. **Configure permissions** (default: OPs only):
- Grant `spectatorplusplus.use` to allow spectator mode
- Grant `spectatorplusplus.admin` for admin commands

### 3. Start Using
1. Players with permission use `/spectator` to toggle spectator mode
2. Use compass (hotbar slot 0) to select players to spectate
3. Right-click players for first-person view
4. Use `/spectator` again to exit
